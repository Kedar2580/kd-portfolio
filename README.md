# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML**, **CSS**, and **Tailwind CSS** to showcase my projects, skills, and contact information. The website is designed with a clean UI, smooth responsiveness, and an engaging layout to provide a professional online presence.

## 🚀 Live Demo

🔗 Live Website: https://kd-portfolio-lilac.vercel.app/

---

# 📌 Project Summary

This portfolio website serves as my personal developer portfolio where visitors can explore my projects, technical skills, and contact details. The website highlights my frontend development abilities through a visually appealing and fully responsive design.

The project focuses on:
- Clean and modern user interface
- Mobile-friendly responsive layout
- Smooth user experience
- Organized project showcase
- Easy navigation and accessibility

It acts as a digital resume and portfolio to represent my work and skills online.
